# Testes Unitários do Registre.Se

## Descrição
Este repositório contém testes unitários de uma ferramenta de registros de usuários.

## Tecnologias Utilizadas
- [ ] Jasmine JS (para testes unitários em JavaScript)
- [ ] JavaScript (linguagem de programação para criar e validar as funções)
- [ ] HTML (para estrutura do formulário de cadastro)
- [ ] CSS (para estilização do formulário)

## Estrutura do Projeto
```
/registre.SE
├── src/       # Projeto a ser testado
├── test/      # Testes de unitários
|
└── index.html  # Página para teste manual
```

## Como Executar os Testes
### 1. Clonar o repositório
```sh
git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio
```

### 2. Instalar Dependências
```sh
https://github.com/jasmine/jasmine/releases
```

### 3. Executar Testes
#### Testes Unitários
```sh
Abra o arquivo SpecRunner.html no diretório test/Specrunner.html
```
